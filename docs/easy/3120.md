# 3120. Count the Number of Special Characters I

## Question

You are given a string `word`. A letter is called **special** if it appears **both** in lowercase and uppercase in `word`.

Return the number of **special** letters in `word`.

### Example 1:
```
Input: word = "aaAbcBC"
Output: 3

Explanation:
The special characters in `word` are `'a'`, `'b'`, and `'c'`.
```

### Example 2:
```
Input: word = "abc"
Output: 0

Explanation:
No character in `word` appears in uppercase.
```

### Example 3:
```
Input: word = "abBCab"
Output: 1

Explanation:
The only special character in `word` is `'b'`.
```

### Constraints:
- `1 <= word.length <= 50`
- `word` consists of only lowercase and uppercase English letters.


## Solution
```dart
class Solution {
  int numberOfSpecialChars(String word) {
    if (word.length <= 1) {
      return 0;
    }

    int count = 0;

    final splitted = word.split('');
    var lowercase = {};
    var uppercase = {};

    splitted.forEach((ele) {
      if (lowercase.containsKey(ele)) {
        lowercase[ele]++;
      } else if (uppercase.containsKey(ele)) {
        uppercase[ele]++;
      } else {
        if (ele == ele.toLowerCase()) {
          lowercase[ele] = 1;
        } else {
          uppercase[ele] = 1;
        }
      }
    });

    lowercase.forEach((key, value) {
      if (uppercase.containsKey(key.toUpperCase())) {
        count++;
      }
    });

    return count;
  }
}
```

## Mindset

Split the input string into an array and iterate through it. For each character, check whether it is lowercase or uppercase, and group it into the corresponding Map. Finally, check for characters that appear in both the `lowercase` Map and the `uppercase` Map, and count them.
