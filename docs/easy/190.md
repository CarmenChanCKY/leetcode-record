# 190. Reverse Bits

## Question
Reverse bits of a given 32 bits signed integer.

### Example 1:
```
Input: n = 43261596
Output: 964176192
```

#### Explanation:
| Integer | Binary |
| --- | --- |
| 43261596 | 00000010100101000001111010011100 |
| 964176192 | 00111001011110000010100101000000 |

### Example 2:
```
Input: n = 2147483644
Output: 1073741822
```

#### Explanation:
| Integer | Binary |
| --- | --- |
| 2147483644 | 01111111111111111111111111111100 |
| 1073741822 | 00111111111111111111111111111110 |

### Constraints
- `0 <= n <= 231 - 2`
- `n` is even.


## Solution
```typescript
function reverseBits(n: number): number {
    let bin = n.toString(2).padStart(32, "0").split("");
    let len = bin.length;
    const halfLen = len / 2;
    len--;

    for (let i = 0; i < halfLen; i++) {
        if (bin[i] !== bin[len - i]) {
            let temp = bin[i]
            bin[i] = bin[len - i]
            bin[len - i] = temp
        }
    }

    const reverse = parseInt(bin.join(""), 2).toString(10);
    return parseInt(reverse);
};
```

## Mindset
1. The given parameter `n` is a number. To convert `n` from decimal to binary, use the JavaScript function `toString(2)`
2. Pad the string with `0` on the left until its length is 32 i.e. 32 bit
3. Split the string to an array of length = `32`
4. Get the middle index of array i.e. $ 32 / 2 = 16 $
5. `len--;`  makes the calculation more convenient because the array index is start from zero. It makes `len` equals to the last index of the array
6. This program swap the position of i<sup>th</sup> element and (len - i)<sup>th</sup> element. Only `array.length / 2` iterations required. So, the for-loop loops from `0` to `array.length / 2`
7. Join the array element to string and convert it back to decimal
