# 796. Rotate String

## Question
Given two strings `s` and `goal`, return `true` *if and only if* `s` *can become* `goal` *after some number of shifts on* `s`.

A **shift** on `s` consists of moving the leftmost character of `s` to the rightmost position.

- For example, if `s = "abcde"`, then it will be `"bcdea"` after one shift.

### Example 1:
```
Input: s = "abcde", goal = "cdeab"
Output: true
```

### Example 2:
```
Input: s = "abcde", goal = "abced"
Output: false
```

### Constraints:
- `1 <= s.length, goal.length <= 100`
- `s` and `goal` consist of lowercase English letters.

## Solution
```java
class Solution {
    public boolean rotateString(String s, String goal) {
        if (s.length() != goal.length()) {
            return false;
        }

        int sLen = s.length();

        StringBuilder sWord = new StringBuilder(s);
        for (int i = 0; i < sLen; i++) {
            if (sWord.toString().equals(goal)) {
                return true;
            }

            char firstChar = sWord.charAt(0);
            sWord.deleteCharAt(0);
            sWord.append(firstChar);
        }
        return false;
    }
}
```

## Mindset
1. If the lengths of strings `s` and `goal` are not equal, they cannot be the same string. So, `s` cannot become `goal`. Return `false` directly.
2. Starting from the first character of string `s`, compare it with string `goal`. If they are the same, return `true`. Otherwise, move the first character of `s` to the end of `s`.
3. Repeat this process until the string becomes the original `s` again. This means `s` cannot become `goal` after any number of shifts, so the function returns `false`.
