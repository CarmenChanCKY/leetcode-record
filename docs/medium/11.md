# 11. Container With Most Water

## Question

You are given an integer array `height` of length `n`. There are `n` vertical lines drawn such that the two endpoints of the `i^th` line are `(i, 0)` and `(i, height[i])`.

Find two lines that together with the x-axis form a container, such that the container contains the most water.

Return *the maximum amount of water a container can store*.

**Notice** that you may not slant the container.

Output: 49

### Example 2:
```
Input: height = [1,1]

```

### Constraints:
- `n == height.length`
- `2 <= n <= 105`
- `0 <= height[i] <= 104`

## Solution
```dart
class Solution {
  int maxArea(List<int> height) {
    int maxArea = 0;
    int len = height.length;

    int ptr1 = 0;
    int ptr2 = len - 1;

    while (ptr1 != ptr2) {
      int area = 0;
      int distance = ptr2 - ptr1;

      if (height[ptr1] < height[ptr2]) {
        area = height[ptr1];
        ptr1++;
      } else {
        area = height[ptr2];
        ptr2--;
      }

      area *= distance;
      if (maxArea < area) {
        maxArea = area;
      }
    }
    return maxArea;
  }
}

```

## Mindset
Create two pointers at the start and end of the array. Compare the values they point to and take the smaller one as the water height. Then calculate the distance between the pointers as the container width. Multiply height by width to get the area, and keep track of the maximum until the pointers meet.
