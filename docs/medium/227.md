# 227. Basic Calculator II

## Question
Given a string `s` which represents an expression, *evaluate this expression and return its value*.

The integer division should truncate toward zero.

You may assume that the given expression is always valid. All intermediate results will be in the range of `[-231, 231 - 1]`.

**Note:** You are not allowed to use any built-in function which evaluates strings as mathematical expressions, such as `eval()`.

### Example 1:
```
Input: s = "3+2*2"
Output: 7
```

### Example 2:

```
Input: s = " 3/2 "
Output: 1
```

### Example 3:

```
Input: s = " 3+5 / 2 "
Output: 5
```

### Constraints:
- `1 <= s.length <= 3 * 105`
- `s` consists of integers and operators `('+', '-', '*', '/')` separated by some number of spaces.
- `s` represents **a valid expression**.
- All the integers in the expression are non-negative integers in the range `[0, 231 - 1]`.
- The answer is **guaranteed** to fit in a **32-bit integer**.


## Solution
```dart
class Solution {
  int calculate(String s) {
    s = s.trim();
    if (int.tryParse(s) != null) {
      return int.parse(s);
    }

    final stack = [];
    final splitStr = [];
    int strLen = s.length;
    String currentNum = "";
    for (int i = 0; i < strLen; i++) {
      if (s[i] == "+" || s[i] == "-" || s[i] == "*" || s[i] == "/") {
        splitStr.add(currentNum);
        splitStr.add(s[i]);
        currentNum = "";
      } else {
        currentNum += s[i];
      }
    }

    if (currentNum.isNotEmpty) {
      splitStr.add(currentNum);
    }

    strLen = splitStr.length;
    for (int i = 0; i < strLen; i++) {
      if (stack.isNotEmpty && (stack.last == "*" || stack.last == "/")) {
        String symbol = stack.removeLast();

        num cal = num.parse(stack.removeLast());

        if (symbol == "*") {
          cal *= num.parse(splitStr[i]);
        } else {
          cal /= num.parse(splitStr[i]);
        }

        stack.add(cal.toInt().toString());
      } else {
        stack.add(splitStr[i]);
      }
    }

    if (stack.length == 1) {
      return num.parse(stack[0]).toInt();
    } else {
      num cal = num.parse(stack[0]);
      for (int i = 1; i < stack.length; i++) {
        if (stack[i] == "+") {
          cal += num.parse(stack[i + 1]);
        } else if (stack[i] == "-") {
          cal -= num.parse(stack[i + 1]);
        }
      }
      return cal.toInt();
    }
  }
}
```

## Mindset
1. Split the input string into symbols and numbers, and store the result in the `splitStr` array. For example, if the input is `" 3+5 / 2 "`, the `splitStr` array should be `["3", "+", "5", "/", "2"]`.
2. Iterate through the `splitStr` array and push each element onto a stack. If the top of the stack is `*` or `/`, pop the operator and the previous number, then multiply or divide it by the current number. Push the result back onto the stack.
3. After the iteration, if the stack contains only one element, return it. Otherwise, iterate through the stack and apply `+` / `-` to accumulate the final result.
