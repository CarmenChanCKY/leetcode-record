# 6. Zigzag Conversion

## Question

The string `"PAYPALISHIRING"` is written in a zigzag pattern on a given number of rows like this: (you may want to display this pattern in a fixed font for better legibility)

```
P   A   H   N
A P L S I I G
Y   I   R
```

And then read line by line: `"PAHNAPLSIIGYIR"`

Write the code that will take a string and make this conversion given a number of rows:

```
string convert(string s, int numRows);
```

### Example 1:
```
Input: s = "PAYPALISHIRING", numRows = 3
Output: "PAHNAPLSIIGYIR"
```

### Example 2:
```
Input: s = "PAYPALISHIRING", numRows = 4
Output: "PINALSIGYAHRPI"

Explanation:
P     I     N
A   L S   I G
Y A   H R
P     I
```

### Example 3:
```
Input: s = "A", numRows = 1
Output: "A"
```

### Constraints:

- `1 <= s.length <= 1000`
- `s` consists of English letters (lower-case and upper-case), `','` and `'.'`.
- `1 <= numRows <= 1000`

## Solution
```dart
class Solution {
  String convert(String s, int numRows) {
    int strLen = s.length;

    if (strLen <= 1 || numRows == 1) {
      return s;
    }

    int middleLength = numRows - 2;

    String result = "";

    for (int i = 0; i < numRows; i++) {
      int count = i;
      bool nextIsMiddle = true;
      while (count < strLen) {
        result += s[count];

        if (i == 0 || i == numRows - 1) {
          count += middleLength + numRows;
        } else {
          if (nextIsMiddle) {
            count += (numRows - middleLength) *
                (i == middleLength ? 1 : middleLength - i + 1);
          } else {
            count += (numRows - middleLength) * i;
          }

          nextIsMiddle = !nextIsMiddle;
        }
      }
    }

    return result;
  }
}

```

## Mindset

1. If the string is empty, contains only one character, or there is only one row, the zigzag conversion does nothing. Return `s`.
2. The number of “middle” rows, which is the diagonal part, is `numRows - 2`.
3. Consider the examples by drawing the zigzag pattern and the index positions for each character:

**Example 1**

```
Input: s = "PAYPALISHIRING", numRows = 3

P   A   H   N     0   4   8     12
A P L S I I G     1 3 5 7 9  11 13
Y   I   R         2   6   10

Return: PAHNAPLSIIGYIR
Print order: 0 4 8 12 1 3 5 7 9 11 13 2 6 10
```

**Example 2**

```
Input: s = "PAYPALISHIRING", numRows = 4

P     I     N     0     6       12
A   L S   I G     1   5 7    11 13
Y A   H R         2 4   8 10
P     I           3     9

Return: PINALSIGYAHRPI
Print order: 0 6 12 1 5 7 11 13 2 4 8 10 3 9
```

**Example 3**

```
Input: s = "PAYPALISHIRING", numRows = 5

P       H      0       8
A     S I      1     7 9
Y   I   R      2   6   10
P L     I G    3 5     11 13
A       N      4       12

Return: PHASIYIRPLIGAN
Print order: 0 8 1 7 9 2 6 10 3 5 11 13 4 12
```

These three examples show the pattern:

**First and last rows**
- The indices for characters in the same row are:
    - First character: `currentRow`
    - Other characters: `previousCharacterIndex + numRows + middleRowLength`

- For example, in Example 3, the index of:

```
middleRowLength = 3
A = currentRow = 4
N = previousCharacterIndex + numRows + middleRowLength = 4 + 5 + 3 = 12
```

**Middle rows**
- The indices for characters in the same row depend on whether the **next** character is inside or outside the diagonal part.
- The first character is always outside the diagonal.
- Characters alternate: outside, inside, outside, inside, and so on.
- As with the first and last rows, the first character in a row starts at `currentRow`.
- Since the first character is outside the diagonal, the next character is inside the diagonal. The index of a character inside the diagonal is:

```
previousCharacterIndex + (numRows - middleLength) * (currentRow == middleLength ? 1 : middleLength - currentRow + 1)
```

- Then the next character goes outside the diagonal again. The index is:

```
previousCharacterIndex + (numRows - middleLength) * currentRow
```

- For example, in Example 3, the indices for:

**Second Row (A S I)**
```
middleRowLength = 3

A = currentRow = 1

S = previousCharacterIndex + (numRows - middleLength) * (currentRow == middleLength ? 1 : middleLength - currentRow + 1)
= 1 + (5 - 3) * (3 - 1 + 1)
= 7 (Since S is inside diagonal and currentRow != middleLength)

I = previousCharacterIndex + (numRows - middleLength) * currentRow
= 7 + (5 - 3) * 1 = 9
```

**Fourth Row (P L I G)**

```
middleRowLength = 3

P = currentRow = 3

L = previousCharacterIndex + (numRows - middleLength) * (currentRow == middleLength ? 1 : middleLength - currentRow + 1)
= 3 + (5 - 3) * 1
= 5 (Since L is inside diagonal and currentRow == middleLength)

I = previousCharacterIndex + (numRows - middleLength) * currentRow
= 5 + (5 - 3) * 3 = 11

G = previousCharacterIndex + (numRows - middleLength) * (currentRow == middleLength ? 1 : middleLength - currentRow + 1)
= 11 + (5 - 3) * 1
= 13
```
