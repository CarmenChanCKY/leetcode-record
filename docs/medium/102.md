# 102. Binary Tree Level Order Traversal

## Question
Given the `root` of a binary tree, return *the level order traversal of its nodes' values*. (i.e., from left to right, level by level).

### Example 1:
![](https://assets.leetcode.com/uploads/2021/02/19/tree1.jpg)

```
Input: root = [3,9,20,null,null,15,7]
Output: [[3],[9,20],[15,7]]
```

### Example 2:
```
Input: root = [1]
Output: [[1]]
```

### Example 3:
```
Input: root = []
Output: []
```

### Constraints:
- The number of nodes in the tree is in the range `[0, 2000]`.
- `1000 <= Node.val <= 1000`

## Solution
```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {

        List<List<Integer>> result = new ArrayList<>();

        if (root != null) {

            Queue<List<TreeNode>> q = new LinkedList<>();

            q.add(List.of(root));
            while (!q.isEmpty()) {
                List<Integer> intGroup = new ArrayList<>();
                List<TreeNode> nodeGroup = q.poll();
                List<TreeNode> newChildGroup = new ArrayList<>();

                for (TreeNode node: nodeGroup) {
                    intGroup.add(node.val);

                    if (node.left != null || node.right != null) {
                        if (node.left != null) {
                            newChildGroup.add(node.left);
                        }

                        if (node.right != null) {
                            newChildGroup.add(node.right);
                        }
                    }
                }

                if (!newChildGroup.isEmpty()) {
                    q.add(newChildGroup);
                }

                if (!intGroup.isEmpty()) {
                    result.add(intGroup);
                }
            }
        }
        return result;
    }
}
```

## Mindset
Use a queue to perform BFS. This queue stores a list containing the nodes at the same level. For every level in the tree, add the nodes’ values and collect their children into a new list. Then add this list to the queue until the queue is empty (i.e., you’ve reached the leaf level).
